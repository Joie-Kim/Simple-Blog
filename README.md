# Simple-blog

## 🤔 Why

- 프론트엔드와 백엔드 간의 통신을 직접 구현해 보고, 이해하기 위해

<br>

## 😎 What

간단한 블로그 애플리케이션을 만들며, CRUD를 경험할 예정입니다.

### Feature

1. **User**

   - 회원가입
   - 로그인 (JWT 인증 방식 사용하여, 로그인 유지)
   - 로그아웃

2. **Post**

   - 블로그 게시글 CRUD
   - Create : 로그인한 사용자만 할 수 있음
   - Read : 모든 사용자 가능
   - Update && Delete : 게시글의 작성자만 할 수 있음

### Skills

- ReactJS
- NodeJS
- Mongo DB
